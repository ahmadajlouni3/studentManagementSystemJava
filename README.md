# Student management system - JAVA

console app with many features like:
- add student
- add note about student
- remove student
- show information about student
- show full docs (NOTES) about specific student

---

## more information about app

the app using modern java Data structure and algorithms to performance operation inside app.

---

## installation

inside app folder write this command

```
javac Main.java
```
to let java compile entire files

after that only run this command
```
java Main
```
and enjoy using this app
